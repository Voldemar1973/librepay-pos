# ⚡️ LibrePay POS Terminal

The official WebApp interface for the **LibrePay** ecosystem. This is a non-custodial, zero-KYC payment terminal designed for merchants in emerging markets.

## 🚀 Features
- **Instant Conversion:** Real-time exchange rates for USD, ARS, and UAH to USDT.
- **Direct-to-Wallet:** Payments go straight to the merchant's TRC-20 address.
- **Privacy First:** No personal data collection or centralized custody.
- **Telegram Integrated:** Built specifically to run as a Telegram WebApp.

- **Network:** TRON (TRC-20)
- **Backend:** Integrated with TronGrid API for blockchain verification.

## 📱 How to use

2. Enter the amount in your local currency.
3. Generate the QR code for the customer.
4. Receive USDT directly into your Trust Wallet.
